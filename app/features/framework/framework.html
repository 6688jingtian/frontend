前端脚手架介绍

快速搭建前端单页应用项目

特性

  • 基于Angular, ui-router 单页应用
  • 基于模块化开发，按照模块按需加载
  • 基于mockserver实现脱离后端独立开发
  • 丰富的UI组件支持及常见页面布局UI设计模版

开发构建

目录结构

文件夹命名说明:

  • app/directives

    可复用组件目录

  • app/features

    功能模块目录，主要以page划分模块

  • app/services

    公用服务模块目录

  • app/style

    基础样式目录，variables.less里定义变量的，可通过修改变量值来定制多套主题

  • app/img

    图片目录

快速开始

克隆脚手架项目

git clone ssh://zhangyou04@icode.baidu.com:8235/baidu/frontend/demo

进入项目根目录（与app同级）安装依赖包

npm run installPkgs（只需安装一次）

启动项目

npm run start

运行项目

脚手架: http://localhost:8888

组件库: http://localhost:8888/dist

开发热部署模式

脚手架: http://localhost:8886(源码路径 app/);

组件库: http://localhost:8887(源码路径: UIComponents/src/)
